# **SE91 - Message Maintenance**

|                  | Description                            |
|-----------------:|----------------------------------------|
|        SAP TCODE | SE91                                   |
|      Description | Message Maintenance                    |
|          Package | SMESSAGE                               |
|     Program Name | RSMESSAGES                             |
|    Screen Number | 1000                                   |
| Transaction Type | R                                      |

## ROLE

Le SAP TCode SE91 est utilisé pour la tâche : Maintenance des messages. Le TCode appartient au package SMESSAGE.
